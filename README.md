# Digital Examination & Proctoring System

A comprehensive full-stack web application for managing digital examinations with built-in proctoring capabilities.

## Tech Stack

- **Database:** Oracle XE
- **Backend:** Node.js + Express.js + oracledb
- **Frontend:** HTML/CSS/JavaScript (Vanilla)

## Project Structure

```
Digital-Examination-Proctoring-System/
├── backend/          # Node.js Express server
├── frontend/         # HTML/CSS/JS frontend
├── sql/              # SQL scripts and schema
├── .gitignore        # Git ignore rules
└── README.md         # This file
```

## Prerequisites

- Node.js (v14 or higher)
- Oracle XE (locally installed)
- Git

## Oracle XE Setup

Before running the project, ensure Oracle XE is installed locally:
- **Username:** system
- **Host:** localhost
- **Port:** 1521
- **Service:** XEPDB1
